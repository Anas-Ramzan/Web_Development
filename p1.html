Control Printing Jobs

Wndows have powerfull features to control printing jobs. Wndows maintain a print
queue for all the jobs to be printed.The default can be changed
at any time by just selecting printers from the setting submenu of the Start
menu,and then setting on the just "set as default printer".

Adding a new printer

You can add a new printer to your computer by following steps:
1.Click the Start
2.Follow the setting
3.Click Printer
